Twitch Watch Streak Alerts

Alerts you when your watch streak increases. Stores streak values per channel locally.

Alerts

Enable Extension

Master on/off switch for everything.

Desktop Notifications

Show a notification when streak increases or is first detected.

Sound Alert

Plays a short beep when the streak increases (and first-detect alert too).

Alert on First Detection

Alerts once per channel when the streak value is detected for the first time.

Auto Open/Close Watch Streak Menu

Automatically clicks the Watch Streak button to open the menu, then closes it after an increase.

Debug HUD Updates

When enabled, HUD updates whenever a streak value is detected.

HUD Visible Time (seconds)

How long the HUD stays visible after an update. Default is 60 seconds.

Tracked Channels

Click a column header to sort. Click again to toggle direction. A down arrow means descending.
